This app will utilize the Spotify and Omdb APIs for node.js

The spotify functionality will allow you to search for songs, artists, and events.

The Omdb will allow you to search for different movies and (hopefully) upcoming ones as well!


Dependancies- Axios, Moment, dotenv, Node-Spotify-API, inquire

I didn't use moment because I couldn't get a key for the bands in town.

API keys needed- OMDB and Spotify
